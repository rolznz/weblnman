# WebLN Manager

[DEMO](https://rolznz.github.io/weblnman/)

## Development

- `yarn install`
- `yarn dev`

Boostrapped with [Vite](https://vitejs.dev/guide/)

## Supported functionality

[x] Show node info

[x] Onchain balance

[x] Show channels

[x] Connect to peer & Open channel

[x] Show invoices

[x] Show transactions
